tackle.parser.providers.get_hook(hook_type, context: Context)[source]¶

Get the hook from available providers.

Does the following to return the hook: 1. Check if hook hasn’t been imported already 2. Check if the hook has been declared in a provider’s __init__.py’s hook_types field. 3. Try to import it then fall back on installing the requirements.txt file
